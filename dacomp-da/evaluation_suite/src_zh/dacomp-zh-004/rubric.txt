## 需求3：探究销冠商品的复购率与销售额之间的关联（最高可得 7分）
### 标准3.1：复购率的定义与计算（最高可得 3分）
#### 路径3.1.A [3分 | 通用定义与计算]
- 小标准3.1.A.1 [1分 | 完备性]: **定义清晰合理**。清晰地阐述了所采用的“复购率”计算公式及其**口径**（例如：购买该商品的顾客中，购买次数 > 1的顾客占比；或者：指定时间窗口内，发生第二次及以上购买行为的顾客占比；其他自定义指标等），逻辑自洽。
- 小标准3.1.A.2 [2分 | 精确性]: **计算过程正确且一致**。其计算流程正确地反映了其**定义和口径**，且覆盖了所有销冠商品。计算应与所采用的定义和口径保持逻辑一致，用于验证的伪代码如下。
 
 ```伪代码：口径一：总购买次数 >1 的顾客占比
  对于 每一个销冠商品:
    1. 筛选出购买过该商品的所有记录。
    2. 按'顾客编号'对记录进行分组，并统计每个顾客的购买次数。
    3. 计算总的独立顾客数 (分母)。
    4. 计算购买次数大于1的独立顾客数 (分子)。
    5. 分子 / 分母 得到该商品的复购率。

  参考对应复购率数据：
  | 销冠产品     | 总金额     | 复购率       |
  | :---------- | :------- | :----------- |
  | DW-1001040125 | 5948.88  | 0.134615385  |
  | DW-2316020016 | 5600     | 0            |
  | DW-1203130446 | 7384.83  | 0.265486726  |
  | DW-1518040045 | 1400.3   | 0.058823529  |
  
 ```伪代码：口径二：指定时间窗口内发生二次及以上购买的顾客占比
    设定时间窗口 T (例如：90天)
    对于 每一个销冠商品:
      1. 筛选出购买过该商品的所有记录。
      2. 按'顾客编号'对记录进行分组。
      3. 对于每个顾客，检查其购买记录中是否存在**两笔或更多**订单，且**最早和最晚**订单的时间间隔**在T以内**。
      4. 计算总的独立顾客数 (分母)。
      5. 计算在时间窗口T内发生二次及以上购买的独立顾客数 (分子)。
      6. 分子 / 分母 得到该商品的复购率。
    ```
 
 ```伪代码：口径三：其他自定义口径
   最终的复购率数据应与所选口径的计算结果保持一致。
  ```
