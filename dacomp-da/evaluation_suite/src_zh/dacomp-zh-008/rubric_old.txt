# [总分 | 30分] 解决方案需满足两个核心需求：
- 需求1：不同项目类型的成本偏差与分布
- 需求2：成本偏差与团队规模/风险等级/客户满意度的联系


## 需求1：不同项目类型的成本偏差与分布（最高 15 分）
目标：在统一统计口径下，对比各项目类型的成本偏差统计与分布形态，识别离群与不稳定类型，并给出可核验结果。

### 标准1.1：口径声明与计算一致性（无数据清洗）（最高 3 分）
#### 路径1.1.A [3分 | 字段映射 + 口径声明 + 复算/复现]
- 小标准1.1.A.1 [1分 | 完备性] 字段映射与指标定义 
- 要点：列出原始→分析字段映射（预算金额、实际成本、项目类型、团队规模、风险等级、客户满意度），显式定义 `cost_dev = 预算金额 − 实际成本`；说明是否额外计算相对偏差 `%`（可选）。 
- 需写明统计范围（如是否含进行中/仅已完成/时间窗）与计量单位/币种及换算。 
- 计分：映射与范围/单位说明齐全 → 1 分；否则 0 分。

- 小标准1.1.A.2 [1分 | 精确性] 计算一致性与分组对齐核验 
- 要点：随机抽样≥5条，展示 budget、actual、cost_dev 的手算与程序结果一致；并对至少 1 个项目类型核对分组样本数与原始记录相符；全表/图单位前后一致。 
- 若涉及随机抽样，需给出固定随机种子（如 `seed=2025`）。 
- 计分：三项均满足 → 1 分；否则 0 分。

- 小标准1.1.A.3 [1分 | 结论性] 口径边界与适用范围声明 
- 要点：解释所选口径对结论方向的潜在影响（示例：含进行中项目通常使 cost_dev 偏正；仅已完成更贴近真实控制；绝对偏差 vs 百分比影响跨规模可比性），并给出**一句话适用范围总结**。 
- 计分：提供影响分析 + 一句话范围总结 → 1 分；否则 0 分。

### 标准1.2：类型分组统计（最高 6 分）
#### 路径1.2.A [6分 | 描述统计 + 一致性核验]
- 小标准1.2.A.1 [2分 | 完备性] 核心统计 6 列 
- 要点：按 `project_type` 计算 **mean / median / std / count / IQR / 95%CI(均值)**，单位统一且列名清晰。 
- 计分：所有类型具备 6 列 → 2 分；≥80% 类型具备且≥4 列 → 1 分；否则 0 分。
- 小标准1.2.A.2 [2分 | 精确性] 复算一致性 
- 要点：任选 ≥1 类型做独立复算（展示中间量/样本数/方差项），与主表结果差异≤1%。 
- 计分：满足阈值 → 2 分；仅口头说明无量化 → 1 分；否则 0 分。
- 小标准1.2.A.3 [2分 | 结论性] 偏斜与稳健性解释 
- 要点：报告每类型 `|mean−median|/std` 并以阈值（建议 0.3）标注“偏斜”；对至少 1 类给出原因与业务解读。 
- 计分：逐类型标注并解释≥1 类 → 2 分；仅总体说明 → 1 分；否则 0 分。

### 标准1.3：分布与离群（最高 3 分）
#### 路径1.3.A [3分 | 直方图/箱线 + IQR 离群表 + 分布结论]
- 小标准1.3.A.1 [1分 | 完备性] 可视与要素 
- 要点：整体直方图（注明分箱法/宽） + 分类型箱线图（中位/IQR/须/离群）；或提供等价统计替代。 
- 计分：两类可视（或等价替代）齐全 → 1 分；否则 0 分。
- 小标准1.3.A.2 [1分 | 精确性] IQR 离群清单与抽检 
- 要点：逐类型用 IQR 规则输出离群清单；抽检任一类型≥3条，手算判定与程序一致。 
- 计分：清单+抽检一致 → 1 分；否则 0 分。
- 小标准1.3.A.3 [1分 | 结论性] 分布形态结论（含判定条件） 
- 要点：在“右偏/左偏/近似对称/重尾”等中选至少一条，并写明触发条件（如“均值>中位数且右尾离群>1%”）。 
- 计分：结论+条件 → 1 分；否则 0 分。

### 标准1.4：类型间显著性与排序（最高 3 分）
#### 路径1.4.A [3分 | 正态/齐性检验 + 主检验 + 事后比较]
- 小标准1.4.A.1 [1分 | 完备性] 方法择优说明 
- 要点：先做正态（Shapiro/QQ）与齐性（Levene）；满足则 ANOVA+Tukey，否则 Kruskal-Wallis+Dunn。 
- 计分：四步链路说明齐全 → 1 分；否则 0 分。
- 小标准1.4.A.2 [1分 | 精确性] 统计量与效应量 
- 要点：报告 F/H、df、p、效应量（η²/ε²）。 
- 计分：四项齐全 → 1 分；否则 0 分。
- 小标准1.4.A.3 [1分 | 结论性] 排序与证据 
- 要点：给出与检验匹配的排序（或“不显著”），并括注两两 p 值/CI。 
- 计分：排序+证据 → 1 分；否则 0 分。

## 需求2：成本偏差与团队规模/风险等级/客户满意度的联系（最高 15 分）

### 标准2.1：相关性与方向一致性（最高 5 分）
#### 路径2.1.A [5分 | Pearson+Spearman 矩阵 + 显著性标注]
- 小标准2.1.A.1 [2分 | 完备性] 两套 3×3 矩阵 
- 要点：对 `{cost_dev, team_size, csat}` 产出 Pearson 与 Spearman 两套 3×3（含 p 值 / 样本量），对角线=1。 
- 计分：两套齐全 → 2 分；仅一套或缺 p 值 → 1 分；否则 0 分。
- 小标准2.1.A.2 [2分 | 精确性] 三对核心关系 
- 要点：分别报告 cost_dev 与 team_size / csat / risk_level 的 r 与 p；risk_level 为类别时可序数化或点双列。 
- 计分：三对齐全且方向判断合理 → 2 分；报告不全或缺显著性 → 1 分；否则 0 分。
- 小标准2.1.A.3 [1分 | 结论性] 方向型阈值结论 
- 要点：按阈值给出“正/负/弱相关/非线性单调”的结论，并标注阈值满足情况。 
- 计分：给出≥1条并标注阈值 → 1 分；否则 0 分。

### 标准2.2：分层/分组与风险等级效应（最高 5 分）
#### 路径2.2.A [5分 | 风险分层箱线 + 均值/CI 表 + 单调性检验]
- 小标准2.2.A.1 [2分 | 完备性] 可视与统计表 
- 要点：对 `risk_level`（低/中/高）给箱线图；输出各档 `mean/median/count/95%CI`；或等价替代。 
- 计分：图+表齐全 → 2 分；其一 → 1 分；否则 0 分。
- 小标准2.2.A.2 [2分 | 精确性] 显著性/趋势 
- 要点：主检验（ANOVA/K-W 或趋势检验）+ 至少一项两两比较或趋势统计（p/效应量）。 
- 计分：主检验+至少一项比较/趋势 → 2 分；仅主检验 → 1 分；否则 0 分。
- 小标准2.2.A.3 [1分 | 结论性] 序关系陈述 
- 要点：给出“中>高>低 / 高>中>低 / （高≈低）<中 / 不显著”等之一，并括注 p/CI/效应量。 
- 计分：序关系+证据 → 1 分；否则 0 分。

### 标准2.3：多变量建模（最高 3 分）
#### 路径2.3.A [3分 | 线性/稳健/分位数回归（三选一）]
- 小标准2.3.A.1 [1分 | 完备性] 模型设定与诊断 
- 要点：模型含 `team_size, csat, risk_level（可序数化）, C(project_type)`；输出系数、SE、t、p、R²/Adj.R²；至少一次诊断（残差/异方差或稳健损失/VIF）。 
- 计分：设定+表格+诊断 → 1 分；否则 0 分。
- 小标准2.3.A.2 [1分 | 精确性] 方向一致性 
- 要点：核心变量方向与 2.1 一致，且≥1 个核心变量显著（p<0.05）。 
- 计分：满足 → 1 分；否则 0 分。
- 小标准2.3.A.3 [1分 | 结论性] 模板化结论 
- 要点：从“规模正效应/满意度负效应/风险正效应/多因子显著/均不显著（并解释）”中择一或多条，括注 β/SE/p（或稳健等价）。 
- 计分：结论+证据 → 1 分；否则 0 分。

### 标准2.4：从统计到业务陈述（最高 2 分）
#### 路径2.4.A [2分 | 证据对齐的业务结论矩阵]
- 小标准2.4.A.1 [1分 | 完备性] 四块覆盖 
- 要点：必须覆盖 1) 项目类型差异；2) 团队规模影响（含阈值/分段）；3) 风险等级差异（均值或波动/离群）；4) 客户满意度关系。 
- 计分：四条齐全 → 1 分；否则 0 分。
- 小标准2.4.A.2 [1分 | 结论性] 指向性 + 统计证据 
- 要点：每条结论需附 **r/p** 或 **β/p** 或 **检验 p/CI** 至少一种；仅点估计不计分。 
- 计分：四条均附统计证据 → 1 分；否则 0 分。
