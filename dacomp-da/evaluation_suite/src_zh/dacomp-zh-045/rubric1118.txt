## 需求1：分析比较高价值用户（钻石/白金）与普通用户在搜索、收藏的品类/品牌偏好，以及搜索活跃时间的差异（本需求最高可得 15分）
### 标准1.1：搜索偏好（品类/品牌）（最高3分）
#### 路径1.1.A [3分 | 频次统计]
- 小标准1.1.A.1 [1分 | 完备性]: 明确群体划分（钻石/白金=高价值；其余=普通）、口径（搜索记录去重/聚合）、度量（品类/品牌搜索占比）。
- 小标准1.1.A.2 [1分 | 精确性]: 按伪流程核验：
```
FOR each 品类/品牌:
计算高价值用户搜索频次/占比 vs 普通用户搜索频次/占比
执行卡方检验或两比例z检验
计算效果量（Cramer's V 或 Cohen's d）
IF p < 0.05: 标记为显著差异
输出Top-N差异清单（品类/品牌名称、频次对比、p值、效果量）
```
- 小标准1.1.A.3 [1分 | 结论性]: 
基于实际数据分析的结论：
1) Top-5显著差异项：高价值用户更偏好牙刷(24次 vs 16次)、苹果手机(21次 vs 14次)、登山包(16次 vs 11次)；普通用户更偏好运动鞋(25次 vs 11次)、羽绒服(20次 vs 14次)；
2) 针对性动作：对高价值用户增投个护、数码产品类定向广告；对普通用户重点推广运动服饰品类；
3) 阈值标准：搜索频次差异≥5次且占比差异≥2%为显著差异。
#### 路径1.1.B [3分 | 偏好分数（TF-IDF/偏好指数）对比]
- 小标准1.1.B.1 [1分 | 完备性]: 给出偏好分数定义（如 log频次×逆人群覆盖）与计算步骤。
- 小标准1.1.B.2 [2分 | 精确性]: 计算偏好分数并验证关键数值（误差≤±0.005）：
参考锚点：高价值用户Top-5偏好项牙刷(0.0430)、苹果手机(0.0376)、登山包(0.0273)、咖啡机(0.0206)、儿童书包(0.0152)。
### 标准1.2：收藏偏好（品类/品牌）差异（最高3分）
#### 路径1.2.A [3分 | 频次/占比差异]
- 小标准1.2.A.1 [1分 | 完备性]: 明确收藏口径（商品层/品牌层），按群体计算占比。
- 小标准1.2.A.2 [1分 | 精确性]: 按伪流程核验：
```
FOR each 品类/品牌:
计算高价值用户收藏频次/占比 vs 普通用户收藏频次/占比
执行卡方检验或两比例检验
计算效果量与置信区间
IF p < 0.05: 标记为显著差异
输出Top-N差异清单（含频次对比、统计量、p值）
```
- 小标准1.2.A.3 [1分 | 结论性]: 
收藏偏好的具体差异：
1) 高价值用户偏好品牌：西藏诺迪康(收藏比率1.21倍)、青岛啤酒(1.13倍)、比亚迪(1.08倍)；
2) 普通用户偏好品牌：云南白药(1.57倍)、贵州茅台(1.38倍)、海尔智家(1.13倍)；
3) 差异化动作：高价值用户货架优先展示医药保健+汽车品类，普通用户运营位重点推广白酒+家电品类，监测收藏转化率目标提升20%。
#### 路径1.2.C [3分 | 收藏取消率与质量差异分析]
- 小标准1.2.C.1 [1分 | 完备性]: 定义取消率=取消收藏数/总收藏数，按群体和品类分别统计。
- 小标准1.2.C.2 [2分 | 精确性]: 计算取消率并验证关键数值（误差≤±2%）：
参考锚点：高价值用户总体取消率46.8% vs 普通用户49.6%，普通用户家电取消率70.6%，高价值用户家电取消率44.0%。
#### 路径1.2.B [3分 | 倾向评分（Propensity/Logit）]
- 小标准1.2.B.1 [1分 | 完备性]: 定义"收藏=1"二分类与特征（群体、品类/品牌哑变量、交互可选）。
- 小标准1.2.B.2 [2分 | 精确性]: 按伪流程核验：
```
构建数据集：y=是否收藏 (0/1)
特征变量：X = [群体哑变量, 品类哑变量, 品牌哑变量, 交互项(可选)]
声明基准类（如普通用户、某个品类为参照）
拟合Logistic回归：logit(P) = β0 + β1*群体 + β2*品类 + β3*品牌
输出结果：OR值、95%置信区间、p值（特别是群体系数显著性）
```
### 标准1.3：搜索活跃时间差异（最高3分）
#### 路径1.3.A [3分 | 时间直方/密度+差异检验]
- 小标准1.3.A.1 [1分 | 完备性]: 定义时间聚合（时段/星期×用户），处理时区/季节性口径。
- 小标准1.3.A.2 [1分 | 精确性]: 按伪流程核验：
```
FOR each 时段(0-23小时 或 周一-周日):
计算高价值用户该时段搜索占比 vs 普通用户该时段搜索占比
执行Kolmogorov-Smirnov检验或比例检验
IF p < 0.05: 标记为显著差异时段
输出活跃时段Top-N与差异程度（含占比对比、检验统计量）
```
- 小标准1.3.A.3 [1分 | 结论性]: 
搜索时间差异的具体发现：
1) 高价值用户活跃峰段：22-23点最活跃(6.0%)、16-17点次活跃(5.6%)，呈现晚间集中特征；
2) 普通用户活跃峰段：00-01点最活跃(8.5%)、08-09点次活跃(6.0%)，呈现深夜+早晨双峰；
3) 触达策略：高价值用户在22-23点推送个护+数码产品，KPI目标打开率>15%；普通用户在00-01点推送运动服饰，转化率目标>8%。
#### 路径1.3.B [3分 | 循环模式（cosinor/傅里叶项）]
- 小标准1.3.B.1 [1分 | 完备性]: 声明模型设定（周期、谐波阶数）与特征构造。
- 小标准1.3.B.2 [2分 | 精确性]: 按伪流程核验：
```
FOR each 群体(高价值/普通):
构造时间特征：sin(2πt/T), cos(2πt/T) (T=24小时或7天)
拟合回归模型：搜索频率 = α + β1*sin + β2*cos + ε
计算幅度：A = √(β1² + β2²)
计算相位：φ = atan2(β1, β2) 
输出显著性检验：F统计量、p值、R²
对比两群体的幅度差异和相位差异
```


- 小标准2.1.A.3 [1分 | 结论性]:

- 预期效果：整体点击率提升20%，高价值用户转化率提升25%，普通用户复购率提升15%
- 监测KPI：给出月度分层转化率、品类渗透率、用户活跃度，每月评估调整策略
#### 路径2.1.B [3分 | 提升度/敏感度驱动]
- 小标准2.1.B.1 [1分 | 完备性]: 定义分层uplift或敏感度（价格/权益/内容）。
- 小标准2.1.B.2 [2分 | 精确性]: 按伪流程核验：
```
FOR each 用户层级:
设计对照实验(随机分组或匹配对照)
计算uplift = 转化率(实验组) - 转化率(对照组)
或计算价格弹性: log(数量) ~ log(价格) + 控制变量
统计检验: t检验或Bootstrap置信区间
输出结果: uplift值、95%CI、p值、样本量
按uplift大小排序用户层级优先级
```

#### 路径2.2.B [3分 | 规则/协同过滤（分层）]
- 小标准2.2.B.1 [1分 | 完备性]: 设定规则（共现/序列）或CF特征（ItemCF/UserCF）并按层计算。

```
选择推荐方法: 关联规则 OR 协同过滤
IF 关联规则:
计算支持度support = P(A∩B), 置信度confidence = P(B|A）
计算提升度lift = confidence/P(B)

IF 协同过滤:

输出分层推荐结果(层级×推荐物品×度量值×排序)
```
